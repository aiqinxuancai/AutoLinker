AutoLinker 核心库函数重写设置

0 条规则

功能说明

静态编译时，AutoLinker 会在链接命令中把规则里的 .lib 放到 krnln_static.lib 前面，并移除命令中同名的绝对路径 .lib 输入项。

配置例子

vc2022 + D:\Libs\CoreOverride.lib 只在当前源码选择的链接器名称包含 vc2022 时生效。

链接器匹配留空表示所有链接器生效；保存为 ForceLinkLib.ini，UTF-8 BOM 与 CRLF。

强制链接规则

路径请填写 32 位 Release .lib 文件；不存在的路径会在编译时被跳过。
